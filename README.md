# Women-Dresses-Reviews-Classifier
مشروعي هو نظام ذكي لتصنيف مراجعات منتجات الملابس النسائية يهدف إلى تصنيف كل مراجعة إلى إحدى ثلاث فئات رئيسية: Dresses وTops وBottoms.

اللغة والبيانات

يستخدم المشروع اللغة الإنجليزية كمجال النصوص.

يعتمد على بيانات معنونة مسبقًا حيث كل مراجعة مرتبطة بفئة صحيحة موجودة في ملف CSV.

النصوص هي المدخل الأساسي للنموذج وتمثل البيانات غير المهيكلة التي تم تحليلها.

الخوارزمية والنموذج

تم استخدام نموذج SVM عبر LinearSVC كخوارزمية رئيسية للتصنيف، نظرًا لفاعليته مع النصوص عالية الأبعاد.

تم تحويل النصوص إلى تمثيل رقمي باستخدام TF-IDF Vectorization لاستخلاص أهم الكلمات والأنماط داخل المراجعات.

الهدف الرئيسي هو تعليم النموذج التمييز بين الفئات الثلاث بدقة عالية.

التدريب والاختبار

تم تقسيم البيانات إلى مجموعة تدريب ومجموعة اختبار لضمان قدرة النموذج على التعميم على مراجعات جديدة.

ركز المشروع على تحقيق دقة موثوقة في التصنيف دون الحاجة لتدخل بشري أثناء الاستخدام الفعلي.

الهدف العام

تم تصميم المشروع ليكون أداة أوتوماتيكية لفهم مراجعات العملاء وتصنيفها بشكل سريع وفعال.

يعكس المشروع تطبيق عملي لمفهوم التعلم تحت الإشراف حيث يتعلم النموذج من الأمثلة المعنونة ليطبق معرفته على بيانات جديدة.

المميزات الأساسية

التعامل مع بيانات نصية كبيرة ومعقدة.

التصنيف الدقيق بين ثلاث فئات رئيسية.

إمكانية توسيع المشروع لاحقًا لإضافة فئات جديدة أو بيانات أكثر.

تطبيق التعلم تحت الإشراف بشكل كامل مع بيانات معنونة مسبقا.
